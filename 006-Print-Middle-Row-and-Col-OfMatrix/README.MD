# 006 - Print Middle Row and Column of Matrix

## 📌 Description

This project demonstrates how to retrieve and print the middle row and the middle column of a two-dimensional matrix in C#.

The program generates a matrix filled with random numbers, displays the entire matrix, then prints the elements of the middle row and the middle column separately.

Example:

Matrix

```
03  07  05
08  01  09
06  04  02
```

Middle Column

```
07
01
04
```

Middle Row

```
08  01  09
```

---

## 🚀 Features

- Generate a matrix with random numbers.
- Print the entire matrix.
- Print the middle row.
- Print the middle column.
- Uses reusable methods for better code organization.

---

## 🧠 Algorithm

1. Create a matrix.
2. Fill the matrix with random numbers.
3. Print the complete matrix.
4. Calculate the middle row index.
5. Calculate the middle column index.
6. Print the elements of the middle row.
7. Print the elements of the middle column.

---

## 📂 Methods

### FillMatrix()

Fills the matrix with random numbers between **1** and **10**.

### PrintMatrix()

Prints the entire matrix.

### PrintMiddleRow()

Prints the elements of the middle row.

### PrintMiddleColumn()

Prints the elements of the middle column.
