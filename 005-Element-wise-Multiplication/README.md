# 005 - Element-wise Multiplication

## 📌 Description

This project demonstrates how to perform element-wise multiplication between two matrices in C#.

The program generates two matrices filled with random numbers, multiplies the corresponding elements, and stores the results in a third matrix.

Example:

Matrix A

```
2  5  3
4  1  7
6  8  2
```

Matrix B

```
3  2  4
5  6  1
7  3  9
```

Result Matrix

```
6   10   12
20   6    7
42  24   18
```

---

## 🚀 Features

- Generate two matrices with random numbers.
- Print both matrices.
- Perform element-wise multiplication.
- Store the result in a new matrix.
- Display the resulting matrix.
- Uses reusable methods for better code organization.

---

## 🧠 Algorithm

1. Create three matrices with the same dimensions.
2. Fill the first and second matrices with random numbers.
3. Print both matrices.
4. Multiply each corresponding element.
5. Store the result in a third matrix.
6. Print the resulting matrix.

---

## 📂 Methods

### FillMatrix()

Fills the matrix with random numbers between **1** and **10**.

### PrintMatrix()

Prints the matrix in a formatted table.

### MultiplyMatrix()

Multiplies the corresponding elements of two matrices and stores the result in a third matrix.
