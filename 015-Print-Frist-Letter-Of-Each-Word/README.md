# 015 - Print First Letter of Each Word

## 📌 Description

This project demonstrates how to extract and print the first letter of every word in a string using C#.

The program reads a sentence from the user, identifies the beginning of each word, and prints its first character.

Example:

Input

```
Welcome To C Sharp Programming
```

Output

```
W
T
C
S
P
```

---

## 🚀 Features

- Read a sentence from the user.
- Detect the beginning of each word.
- Print the first letter of every word.
- Uses a Boolean flag to track word boundaries.
- Simple and efficient string traversal.

---

## 🧠 Algorithm

1. Read a string from the user.
2. Set a Boolean variable to indicate the start of a word.
3. Traverse the string character by character.
4. If the current character is not a space and marks the beginning of a word, print it.
5. Update the Boolean flag based on whether the current character is a space.
6. Continue until the end of the string.

---

## 📂 Methods

### ReadString()

Prompts the user to enter a string and returns the input.

### PrintFirstLetterOfEachWord()

Traverses the string and prints the first letter of each word.
