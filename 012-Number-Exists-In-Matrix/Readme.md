# 012 - Number Exists in Matrix

## 📌 Description

This project demonstrates how to search for a specific number in a two-dimensional matrix using C#.

The program traverses the matrix element by element and determines whether the target number exists. It returns a Boolean result indicating whether the number was found.

Example:

Matrix

```
77   5   12
22  20    1
 1   0    9
```

Target Number

```
77
```

Result

```
Success: The number 77 was found in the matrix.
```

---

## 🚀 Features

- Print the matrix.
- Search for a specific number.
- Return a Boolean result (`true` or `false`).
- Display a clear success or failure message.
- Uses reusable methods for better code organization.

---

## 🧠 Algorithm

1. Create a two-dimensional matrix.
2. Specify the target number.
3. Traverse every element in the matrix.
4. Compare each element with the target number.
5. If a match is found, return **true**.
6. If the search completes without finding a match, return **false**.

---

## 📂 Methods

### PrintMatrix()

Prints the matrix in a readable format.

### IsNumberInMatrix()

Searches the matrix for a specified number and returns `true` if found; otherwise, returns `false`.
