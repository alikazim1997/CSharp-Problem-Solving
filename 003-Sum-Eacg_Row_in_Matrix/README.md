# 003 - Sum Each Row in Matrix

## 📌 Description

This project demonstrates how to calculate the sum of each row in a two-dimensional matrix using C#.

The program generates a matrix filled with random numbers, prints the matrix, and then calculates and displays the sum of every row.

Example:

Matrix

```
12  45  31
78  15  20
56  90  14
```

Row Sums

```
Row 1 = 88
Row 2 = 113
Row 3 = 160
```

---

## 🚀 Features

- Fill the matrix with random numbers.
- Print the generated matrix.
- Calculate the sum of each row.
- Display the sum for every row separately.
- Uses reusable methods for better code organization.

---

## 🧠 Algorithm

1. Create a two-dimensional matrix.
2. Fill the matrix with random numbers.
3. Print the matrix.
4. Loop through each row.
5. Sum all elements in the current row.
6. Print the result before moving to the next row.

---

## 📂 Methods

### FillArray()

Fills the matrix with random numbers between **1** and **100**.

### PrintArray()

Prints the matrix in a readable format.

### Sum()

Calculates and returns the sum of a specific row.

### Sum_Arr()

Loops through all rows and prints the sum of each one.
