# Fill Matrix with Random Numbers in C#

## 📌 Description

This project demonstrates how to fill a two-dimensional array (matrix) with random numbers in C# and display the result in the console.

Each element in the matrix is assigned a random integer between **1** and **100**.

---

## 🚀 Features

- Generate random numbers between **1** and **100**.
- Fill any matrix size (rows × columns).
- Print the matrix in a readable format.
- Uses a single `Random` instance for better randomness.

---

## 🧠 Algorithm

1. Create a matrix with the desired dimensions.
2. Generate a random number for each cell.
3. Store the number in the matrix.
4. Print the completed matrix.

---

## 📂 Methods

### `_GetRandomNumber()`

Returns a random integer between **1** and **100**.

### `Full_Arr_Number()`

Fills every element of the matrix with random values.

### `Print_Arr_Number()`

Displays the matrix in the console.

---

## 💻 Example Output

```
15    82    41
67    9     95
```

> The output changes every time because the values are generated randomly.
