# 011 - Count Number in Matrix

## 📌 Description

This project demonstrates how to count the number of occurrences of a specific value in a two-dimensional matrix using C#.

The program traverses every element in the matrix, compares it with the target number, and returns the total number of matches.

Example:

Matrix

```
9  1  12
0  9  1
0  9  9
```

Target Number

```
9
```

Result

```
Occurrences: 4
```

---

## 🚀 Features

- Print the matrix.
- Search for a specific number.
- Count all occurrences of the target number.
- Return the total count.
- Uses reusable methods for better code organization.

---

## 🧠 Algorithm

1. Create a two-dimensional matrix.
2. Specify the target number.
3. Traverse every element in the matrix.
4. Compare each element with the target number.
5. Increment a counter whenever a match is found.
6. Return and display the total number of occurrences.

---

## 📂 Methods

### Print()

Prints the matrix in a readable format.

### CountNumberInMatrix()

Counts how many times a specified number appears in the matrix and returns the result.
