# 014 - Min & Max Number in Matrix

## 📌 Description

This project demonstrates how to find the minimum and maximum values in a two-dimensional matrix using C#.

The program traverses every element in the matrix, determines the smallest and largest values, and displays the results.

Example:

Matrix

```
77   5   12
22  20    6
14   3    9
```

Result

```
Minimum Number is: 3
Maximum Number is: 77
```

---

## 🚀 Features

- Print the matrix.
- Find the minimum value.
- Find the maximum value.
- Display the results with colored console output.
- Uses reusable methods for better code organization.

---

## 🧠 Algorithm

1. Create a two-dimensional matrix.
2. Print the matrix.
3. Initialize the minimum and maximum values with the first matrix element.
4. Traverse every element in the matrix.
5. Update the minimum value whenever a smaller element is found.
6. Update the maximum value whenever a larger element is found.
7. Display the final minimum and maximum values.

---

## 📂 Methods

### PrintMatrix()

Prints the matrix in a readable format.

### MinNumberInMatrix()

Finds and returns the smallest value in the matrix.

### MaxNumberInMatrix()

Finds and returns the largest value in the matrix.
