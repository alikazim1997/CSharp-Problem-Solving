# 004 - Sum Each Row in Matrix In Array

## 📌 Description

This project demonstrates how to calculate the sum of each row in a two-dimensional matrix and store the results in a one-dimensional array using C#.

The program generates a matrix filled with random numbers, calculates the sum of every row, stores each sum in a separate array, and then prints the results.

Example:

Matrix

```
15  28  10
41  36  52
90  17  23
```

Row Sums Array

```
53
129
130
```

---

## 🚀 Features

- Fill the matrix with random numbers.
- Print the generated matrix.
- Calculate the sum of each row.
- Store all row sums in a one-dimensional array.
- Print the row sums from the array.
- Uses reusable methods for better code organization.

---

## 🧠 Algorithm

1. Create a two-dimensional matrix.
2. Create a one-dimensional array to store row sums.
3. Fill the matrix with random numbers.
4. Print the matrix.
5. Calculate the sum of each row.
6. Store each sum in the array.
7. Print all stored sums.

---

## 📂 Methods

### FillArray()

Fills the matrix with random numbers between **1** and **100**.

### PrintArray()

Prints the matrix in a readable format.

### GetRowSum()

Calculates and returns the sum of a specific row.

### CalculateAndStoreRowSums()

Calculates the sum of every row and stores the results in a one-dimensional array.

### PrintArraySums()

Prints all stored row sums.
